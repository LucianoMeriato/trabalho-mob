# Atacadão — Gestão de Produtos

App mobile feito em **React Native + Expo** para cadastrar e gerenciar produtos de um atacadão.

## O que o app faz

- Login e cadastro de usuários (Firebase Auth)
- Recuperação de senha por e-mail
- Cadastrar, editar, listar e excluir produtos (Firebase Realtime Database)
- Ler código de barras pela câmera do celular
- Formatação de preço em R$

## Tecnologias

- React Native + Expo
- Firebase (Auth + Realtime Database)
- React Navigation
- Expo Camera

## Como rodar

```bash
npm install
npx expo start
```

Depois é só escanear o QR Code com o app **Expo Go** no celular.
